# Impact Model

## Objective

Estimate measurable impact from end-to-end automation of intelligence production and packaging.

## Baseline vs AURIXA model

Assumptions for one business-news intelligence cycle:

- Baseline manual flow:
  - Source reading and bullet prep: 18 minutes
  - Compliance and editorial review: 22 minutes
  - Vernacular adaptation: 16 minutes
  - Video brief prep: 20 minutes
  - Total: 76 minutes

- AURIXA assisted flow:
  - Ingestion + drafting + extraction: 4 minutes
  - Compliance + editor loop + approval: 8 minutes
  - Vernacular output in same pass: 3 minutes
  - Video brief generation + studio review: 7 minutes
  - Total: 22 minutes

## Time reduction

- Time saved per cycle = 76 - 22 = 54 minutes
- Reduction percentage = 54 / 76 = 71.1%

## Throughput impact example

Assume 30 cycles per workday:

- Baseline effort = 30 x 76 = 2280 minutes (38.0 hours)
- AURIXA effort = 30 x 22 = 660 minutes (11.0 hours)
- Savings = 1620 minutes (27.0 hours/day)

## Consistency and governance impact

- Single canonical state object reduces cross-channel drift.
- Automated risk and confidence scoring improves review prioritization.
- Audit trail enables post-incident traceability for every run.

## Financial proxy (example)

Assume blended operations cost = INR 1200/hour:

- Daily hours saved: 27.0
- Daily value = 27.0 x 1200 = INR 32,400
- Monthly value (22 workdays) = INR 712,800

- These are directional estimates for hackathon evaluation.
- Replace assumptions with observed run metrics during final pitch.
